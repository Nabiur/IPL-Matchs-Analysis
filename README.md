## Overview
This project explores Indian Premier League (IPL) match data using Python and Jupyter Notebook. The analysis focuses on team performance, venue behavior, toss impact, and season-level trends.

## Repository structure
- `IPL_matches.ipynb`: main notebook for exploratory analysis and visualizations.
- `matches.csv`: source dataset used by the notebook.
- `demo`: supporting project asset.
- `README.md`: project documentation.

## Key analysis areas
- Data cleaning and preprocessing.
- Team-level win and loss patterns.
- Toss decision impact on match outcomes.
- Venue-wise performance trends.
- Comparative performance over seasons.

## Tech stack
- Python
- Jupyter Notebook
- pandas, numpy
- matplotlib, seaborn

## How to run
1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install pandas numpy matplotlib seaborn jupyter
   ```
3. Launch the notebook:
   ```bash
   jupyter notebook IPL_matches.ipynb
   ```
4. Run all cells to reproduce charts and insights.

## Example insights this project can produce
- Most consistent winning teams.
- Stadiums with strong home/away patterns.
- Whether toss choice influences results.
